# contributing to rbtracker

pull requests are welcome. rbtracker is solo-maintained, so please bear with response times.

## getting started

1. fork the repo
2. clone your fork
3. open `RedBullTrackeriOS.xcodeproj` in Xcode
4. set up your `AppConfig.plist` with your own Appwrite project (see README)
5. make your changes on a feature branch

## pull requests

- open an issue first to discuss what you want to change (unless it's a tiny fix)
- keep PRs focused — one thing per PR
- test your changes on an iOS 17+ simulator or device
- update the README if your change affects setup or usage

## code style

- follow standard Swift conventions
- use SwiftUI patterns already established in the codebase
- keep views small and composable
